# Top Cheap Products Magento 2 Module


## Install with Composer

Add the next code in composer.json:

    "require-dev": {
        "gmc/module-top-cheap-products": "dev-master"
    },
    "repositories": [
          "type": "git",
          "url": "git@github.com:mariuscris/top-cheap-products.git"
        }
    ]
